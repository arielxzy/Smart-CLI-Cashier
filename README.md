# 🛒 Smart-CLI-Cashier

Aplikasi Kasir berbasis **CLI (Command Line Interface)** interaktif yang dibangun menggunakan **Node.js**. Repositori ini merupakan **proyek pembelajaran pribadi** yang dikembangkan untuk mengeksplorasi, bereksperimen, dan menguasai berbagai konsep fundamental JavaScript modern.

## 🚀 Fitur Utama
- **Manajemen Inventori Dinamis**: Mengatur data barang toko secara rapi menggunakan struktur data modern.
- **Mutasi Data Real-time**: Stok barang otomatis terpotong secara akurat setiap kali transaksi berhasil dilakukan.
- **Sistem Rekursif Pintar**: Otomatis menanyakan opsi ulang (ganti barang atau tetap coba dengan sisa stok) jika jumlah pembelian melebihi stok tersedia tanpa membuat aplikasi *crash*.
- **Struk Belanja Estetik**: Menampilkan detail pembelian, total harga, hingga sisa stok langsung di dalam terminal.

## 🧠 Kompetensi & Fondasi yang Dikuasai
Melalui proyek praktik langsung ini, saya telah berhasil menerapkan berbagai konsep dasar hingga tingkat lanjut dalam JavaScript:
- [x] **Variabel Scope**: Penggunaan `const` dan `let` secara tepat untuk menghindari kebocoran data dan menjaga efisiensi memori.
- [x] **Tipe Data**: Manipulasi data *String*, *Number*, dan *Boolean* (termasuk konversi tipe data dinamis menggunakan `Number()`).
- [x] **Operator**: Penggunaan operator Aritmatika (`-=`, `*`), Perbandingan (`===`), dan Logika Kebalikan (`!`).
- [x] **Template Literals**: Menyusun teks terminal dinamis menggunakan simbol backtick (`` ` ``).
- [x] **Control Flow**: Mengontrol alur logika aplikasi menggunakan percabangan `if / else` di dalam perulangan.
- [x] **Data Structures**: Mengelola sekumpulan data yang kompleks menggunakan kombinasi *Array of Objects*.
- [x] **Asynchronous & Rekursif**: Menangani input interaktif terminal `readline` menggunakan fungsi rekursif (fungsi yang memanggil dirinya sendiri).

## 🛠️ Cara Menjalankan Aplikasi
1. Pastikan kamu sudah menginstal **Node.js** di perangkat komputer/laptopmu.
2. Buka terminal di folder proyek ini, lalu jalankan perintah:
```bash
node cashier.js
```
